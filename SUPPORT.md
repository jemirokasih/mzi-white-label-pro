# Support

For setup help, bug reports, and feature requests, use the GitHub repository issues or contact PT Mikrotek Zemiro Indonesia.

Website: https://mzi.co.id

Before reporting a bug, please include:

- WordPress version.
- PHP version.
- MZI White Label Pro version.
- Active theme and relevant plugins.
- Clear steps to reproduce the issue.

For production websites, test plugin updates on staging before installing them on a live client site.
