# Security Policy

## Supported Versions

Security fixes are provided for the latest released version of MZI White Label Pro.

| Version | Supported |
| --- | --- |
| 3.x | Yes |
| 2.x | No |

## Reporting a Vulnerability

Please report security issues privately to PT Mikrotek Zemiro Indonesia before public disclosure.

Website: https://mzi.co.id

Include:

- A clear description of the issue.
- Steps to reproduce.
- WordPress, PHP, and plugin versions.
- Any relevant logs or screenshots.

Do not open public GitHub issues for security vulnerabilities.
